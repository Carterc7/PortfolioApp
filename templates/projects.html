Mock O' Clock

Mock O' Clock is a full-stack Java Spring Boot web app for fantasy football fans. Users can view Average Draft Positions (ADPs), explore detailed player statistics, and run customizable mock drafts against intelligent CPU opponents — perfect for testing draft strategies before the real thing.

PollMatic

PollMatic is a full-stack Go web app that makes it easy to create, share, and participate in polls. Users can create custom polls with multiple options and share them publicly or privately via a unique key. While anyone can vote without an account, registered users can manage their polls — making it perfect for quick feedback, group decisions, or anonymous input.

Mock O' Clock is a comprehensive fantasy football application designed to help users prepare for their drafts. Built with Java Spring Boot, it provides a robust platform for viewing player statistics, exploring Average Draft Positions (ADPs), and participating in mock drafts against intelligent CPU opponents.

The application features a clean, intuitive interface that makes it easy to navigate through player data, compare statistics, and test different draft strategies. Whether you're a seasoned fantasy player or just getting started, Mock O' Clock provides the tools you need to make informed decisions and improve your draft performance.

With real-time data updates and customizable draft settings, users can simulate various draft scenarios and practice their strategies before the actual draft day. The platform serves as an essential tool for fantasy football enthusiasts looking to gain a competitive edge.
